# Problem
Given

nums = [2,7,11,15]
target = 9

Return indices.

Example

Output:
[0,1]

Difficulty:
⭐ Easy

## Solution
```python
n = int(input())

arr = []

for i in range(n):
    arr.append(int(input()))

target = int(input())


def find(current_sum, i, list_sum):

    if i >= len(arr):
        return False

    # choose current element
    current_sum += arr[i]
    list_sum.append(arr[i])

    if current_sum == target:
        print(list_sum)
        return True

    if find(current_sum, i + 1, list_sum):
        return True

    # backtrack
    list_sum.pop()

    # skip current element
    if find(current_sum - arr[i], i + 1, list_sum):
        return True

    return False


find(0, 0, [])

```
