# Object-Oriented Programming (OOP)
## Class

A blueprint for creating objects.

## Object

An instance of a class.

## Inheritance

One class acquires the properties and methods of another class.

## Polymorphism

The same method behaves differently depending on the object.

Example:
```Java
Animal a = new Dog();
a.sound();
```

## Encapsulation

Wrapping data and methods together while restricting direct access.
```Java
private int age;

public void setAge(int age){
    this.age = age;
}
```

## Abstraction

Hiding implementation details and showing only essential features.
```Java
abstract class Animal{
    abstract void sound();
}
```

## Constructor

A special method that is automatically called when an object is created.
