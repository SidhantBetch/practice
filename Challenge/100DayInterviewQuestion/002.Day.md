# Developer & Data Science Interview Questions (Short Answers)

## Python

### 1. What is Python?

**Answer:** Python is a high-level, interpreted, object-oriented
programming language.

### 2. List vs Tuple

  List      Tuple
  --------- -----------
  Mutable   Immutable
  \[\]      ()
  Slower    Faster

### 3. Dictionary

**Answer:** Stores data as key-value pairs.

### 4. Set

**Answer:** Stores unique elements and removes duplicates.

### 5. `==` vs `is`

-   `==` compares values.
-   `is` compares memory locations.

### 6. What is OOP?

**Answer:** Programming using classes and objects.

**Four pillars:** Encapsulation, Inheritance, Polymorphism, Abstraction.

### 7. Inheritance

**Answer:** One class acquires properties of another.

### 8. Polymorphism

**Answer:** Same method behaves differently for different objects.

### 9. Encapsulation

**Answer:** Wraps data and methods together and hides data.

### 10. Abstraction

**Answer:** Hides implementation and shows only essential features.

### 11. `append()` vs `extend()`

-   `append()` adds one item.
-   `extend()` adds multiple items.

### 12. Lambda Function

**Answer:** Anonymous one-line function.

### 13. Exception Handling

**Answer:** Uses `try`, `except`, and `finally`.

### 14. `break` vs `continue`

-   `break` stops the loop.
-   `continue` skips the current iteration.

## DSA

### 15. Time Complexity

Measures execution time as input grows.

### 16. Space Complexity

Memory used by an algorithm.

### 17. Big-O

Worst-case complexity (O(1), O(log n), O(n), O(n log n), O(n²)).

### 18. Fastest Sorting

Quick Sort average: **O(n log n)**.

### 19. Stack vs Queue

-   Stack: LIFO
-   Queue: FIFO

### 20. Binary Search

Time: **O(log n)** (sorted array only).

### 21. Merge Sort

Time: **O(n log n)**, Space: **O(n)**.

### 22. DFS vs BFS

-   DFS → Stack
-   BFS → Queue

### 23. Array vs Linked List

-   Array: Fast access, fixed size.
-   Linked List: Dynamic, easy insertion/deletion.

### 24. Recursion

Function calling itself.

### 25. HashMap

Key-value storage. Average search: **O(1)**.

## SQL

### 26. What is SQL?

Language to manage databases.

### 27. DELETE vs TRUNCATE vs DROP

-   DELETE: Removes selected rows.
-   TRUNCATE: Removes all rows.
-   DROP: Deletes the table.

### 28. Primary Key

Unique and NOT NULL.

### 29. Foreign Key

Connects two tables.

### 30. JOIN

Combines data from multiple tables.

### 31. WHERE vs HAVING

-   WHERE: Before GROUP BY.
-   HAVING: After GROUP BY.

### 32. GROUP BY

Groups similar values.

### 33. COUNT(\*) vs COUNT(column)

-   COUNT(\*): All rows.
-   COUNT(column): Ignores NULL.
